This year the qualification round consisted of 4 problems. As lately I have been using Ruby I decided to try my luck with it, and avoid writing all the Java boilerplate code to do simple things like reading and parsing a text file. I am using Ubuntu 12.04 installed in VirtualBox 4.1.12

The competition started at 01:00 AM in my country, but nevertheless I stayed up and tried (and solved) at least one of the problems before sleep kicked in. These are the solutions I came up with under the circumstances.

Problem A: Speaking in Tongues

This one was the first I tried, and in my opinion the easiest of the set. The explanation and the hints made very clear what was to do. This solution gave me 15 points:

#googlerese to english
mapping =
  {"a" => "y", "b" => "h", "c" => "e", "d" => "s", "e" => "o",
   "f" => "c", "g" => "v", "h" => "x", "i" => "d", "j" => "u",
   "k" => "i", "l" => "g", "m" => "l", "n" => "b", "o" => "k",
   "p" => "r", "q" => "z", "r" => "t", "s" => "n", "t" => "w",
   "u" => "j", "v" => "p", "w" => "f", "x" => "m", "y" => "a",
   "z" => "q"}

file = File.new("input", "r")
max = file.gets
counter = 1

while (line = file.gets)
  translatedLine = String.new
  line.split("").each do |i|

    if (i == " ")
        translatedLine << " "
    else
        translatedLine << mapping[i].to_s
    end
  end

  puts "Case ##{counter}: #{translatedLine}"
  counter = counter + 1
end
file.close
